\subsubsection*{template$<$typename T$>$\newline
class MODEL\+::\+Block$<$ T $>$}
Base class for 0D model components. 

A \mbox{\hyperlink{class_m_o_d_e_l_1_1_block}{MODEL\+::\+Block}} is the base class of 0D model elements. It is the place where the contribution of an element to the global system is controlled. It defines all relevant attributes and methods of an element and a few common helpers for setting it up.

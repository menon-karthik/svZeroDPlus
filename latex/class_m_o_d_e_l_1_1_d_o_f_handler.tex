\doxysubsection{Detailed Description}
Degree-\/of-\/freedom handler. 

This class handles degrees-\/of-\/freedom for model variables and equations. It assigns each element with row and column indices which it can use to assemble it\textquotesingle{}s local contributions into the global system.
